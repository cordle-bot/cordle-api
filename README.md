# **`cordle-api`**

API for Cordle Bot

## *`Modules`*

- [Gin](https://github.com/gin-gonic/gin)
- [Gorm](https://github.com/go-gorm/gorm)

## *`Usage`*

### *`Running`*

1. `$ go mod download`

2. Ensure you have a `.env` in the project root following `.env.example` as a guide.

3. `$ make run`
